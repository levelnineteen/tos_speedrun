# tos_speedrun
##機能
マップ移動もしくはコマンド使用時に、区間にかかった時間、シルバーの増減、獲得経験値を表示する。
##使い方
TreeofSaviorJP\data の中にipfファイルを設置する。  
新しいマップに入るか、/speedrun コマンドで、記録をシステムメッセージで表示。  
コマンドを使用した場合、バラックへ戻る。  
TreeofSaviorJP\addons\speedrun\record.csv フォルダと空のファイルを作って設置すると、記録が書き込まれる。
